# Context root
```
/piston-service/v2.0/
```

## User store
### Getting group membership for a user
```
GET /userStore//users/{userIdentifier}/membership
```

### Searching users
```
GET /userStore/users/externalIds
```

### Searching groups
```
GET /userStore/groups/externalIds
```

### Getting user metadata
```
GET /userStore/users/{uid}/metadata
```
